# AdjNoun-Generator

## Parameters

```-1```

(adjective)+(noun)+(1 digit) only

```-2```

(adjective)+(noun)+(2 digits) only

```-3```


adjective)+(noun)+(3 digits) only

```-full```

968 adjectives and 1,844 nouns


```-write```

Writes the output to a file (output.txt)


Default: 153 adjectives and 136 nouns
